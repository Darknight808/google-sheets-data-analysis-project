# 📊 Google Sheets Data Analysis Project

## 📌 Project Overview
This project is a data analysis and reporting project built using Google Sheets.

The goal of this project is to clean, analyze, and visualize data using spreadsheet tools, and to extract meaningful insights for decision-making.
